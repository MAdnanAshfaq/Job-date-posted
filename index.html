How dates are hidden

01
JSON-LD Schema
Most job sites embed a hidden <script type="application/ld+json"> block with a JobPosting schema containing "datePosted". Invisible on the page, readable in source. This is what Google Jobs uses.
datePosted · datePublished
02
Meta Tags
Sites often include <meta> tags for SEO with publish dates. Common on news sites and job boards that repurpose article schemas.
article:published_time · pubdate · DC.date
03
Microdata
Older structured data standard. Dates embedded directly as HTML attributes with itemprop or <time datetime> tags anywhere in the markup.
itemprop=datePosted · <time datetime>
04
URL Pattern
Some job boards encode the date in the URL path itself (e.g. /jobs/2026/04/05/...). Last resort — least reliable, but sometimes the only signal available.
Path segments · query params

Fetches via CORS proxy. Works on static HTML pages. JS-rendered sites (Greenhouse, Workday) may not return results.
