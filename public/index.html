How dates are found

01

JSON-LD structured data

Most reliable. Sites embed a hidden <script type="application/ld+json"> block with a JobPosting schema containing "datePosted". Invisible on the rendered page but present in raw HTML source. Google uses this for job search cards.

"datePosted": "2026-04-05"
02

HTML meta tags

Many CMS platforms write the publish date into <meta> tags for SEO or Open Graph. Common ones: article:published_time, og:published_time, meta name="date".

<meta property="article:published_time" content="2026-04-05T10:00:00Z">
03

Microdata / time elements

Older Schema.org format using HTML attributes directly. Elements with itemprop="datePosted" or <time datetime="..."> tags embedded in the markup itself.

<time itemprop="datePosted" datetime="2026-04-05">
04

URL date patterns

Last resort. Some job portals embed the post date directly in the URL path or query string, e.g. /jobs/2026/04/05/data-engineer or ?date=2026-04-05.

/jobs/2026/04/05/senior-engineer

Reads raw HTML source via a serverless proxy — same technique used by job aggregators.
